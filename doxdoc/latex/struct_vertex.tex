\doxysubsection{Descrição detalhada}
Representa um vértice (nó) no grafo. 

Cada vértice possui um identificador único, uma posição na matriz (\doxylink{struct_vector2}{Vector2}), um valor (carácter representando o tipo, como \textquotesingle{}A\textquotesingle{} ou \textquotesingle{}B\textquotesingle{}), uma lista de arestas (ligações), uma flag de controlo de visitação (seen) e um ponteiro para o próximo vértice.
